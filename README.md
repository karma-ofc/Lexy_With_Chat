# Lexy with Chat

Приложение для изучения иностранных языков с использованием метода интервального повторения и функцией коллективного чата.

## Возможности

- Регистрация и авторизация пользователей с использованием JWT
- Создание и управление колодами карточек со словами
- Система интервального повторения для эффективного запоминания
- Отслеживание статистики обучения
- Режим учёбы с интерактивными карточками
- Публичные колоды для поиска и использования
- Реал-тайм чат между пользователями на базе WebSocket (Socket.IO)
- Групповые чаты и прямые сообщения
- Шифрование сообщений в чате
- Адаптивный интерфейс для всех устройств

## Технический стек

- **Frontend:** React 19 + Vite
- **Backend:** Node.js + Express.js
- **Real-time:** Socket.IO (WebSocket)
- **База данных:** PostgreSQL (две базы: lexy, chat)
- **Контейнеризация:** Docker + Docker Compose
- **Аутентификация:** JWT (JSON Web Tokens)

## Быстрый старт

```bash
docker compose up --build
```

Приложение будет доступно по адресу `http://localhost:5173`
