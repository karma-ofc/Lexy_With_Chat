# Установка и быстрый старт (Docker)

Рекомендуется использовать Docker Compose для локального развёртывания.

1. Клонировать репозиторий

```bash
git clone https://github.com/karma-ofc/Lexy_With_Chat.git
cd Lexy_With_Chat
```

2. Запустить контейнеры

```bash
docker compose up --build
```

Frontend будет доступен на `http://localhost:5173`.
