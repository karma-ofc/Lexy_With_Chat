# Обзор схемы базы данных

Краткое описание основных таблиц в базе `lexy` и `chat`.

`lexy`:
- `users` — пользователи
- `decks` — колоды карточек
- `cards` — карточки в колодах
- `stats` — статистика обучения

`chat`:
- `threads` — потоки сообщений (диалоги и групповые)
- `messages` — сообщения
- `groups` — группы/чаты

Полная схема доступна в SQL-скриптах миграций.
