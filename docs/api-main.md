# API: Main Server

Список основных REST-эндпоинтов основного сервера (`/api`).

Примеры:

- `GET /api/health` — проверка статуса
- `POST /api/auth/login` — логин
- `POST /api/auth/register` — регистрация
- `GET /api/users/:id` — информация о пользователе
- `GET /api/decks` — список колод
- `POST /api/decks` — создать колоду
- `GET /api/cards` — список карточек

Документация живёт в этом файле и может дополняться по мере рефакторинга.
